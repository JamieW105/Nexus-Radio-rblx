Support & Setup

Get help with installation and configuration

Discord Token Setup

User Token (Recommended):

  1. Open Discord in your web browser
  2. Press F12 to open Developer Tools
  3. Go to Application → Local Storage → discord.com
  4. Find "token" and copy its value

Bot Token:

  1. Visit Discord Developer Portal
  2. Create a new application
  3. Go to Bot section and create a bot
  4. Copy the bot token

System Requirements

  • OS: Windows 10/11 (64-bit)
  • RAM: 4 GB minimum
  • Storage: 300 MB free space
  • Audio: Microphone and speakers/headset
  • Network: Internet connection

Troubleshooting

Common Issues:

  • Ensure your Discord token is valid
  • Check firewall settings for network access
  • Verify microphone permissions
  • Use windowed mode for overlay compatibility
